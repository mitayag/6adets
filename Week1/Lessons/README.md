## ðŸ§  Lesson Overview

Mobile computing refers to the ability to transmit **data, voice, and video** through wireless-enabled devices such as smartphones, tablets, and laptops.  
It enables users to access network services and applications **anytime, anywhere**, breaking free from the limitations of wired infrastructure.

## ðŸ§± 2. Software Classification

### ðŸ§© System Software
Software designed to operate and control the hardware directly.  
- Examples: Android, iOS, Windows, Linux  
- Functions: Manages resources, handles device communication, and provides a platform for applications.

### ðŸ’¼ Application Software
Software that helps users perform specific tasks.  
- Examples: Messaging Apps, Mobile Games, Browsers, Social Media  
- Functions: Provides end-user functionality built on top of system software.

## ðŸ¤– 4. Introduction to Android Operating System

### What is Android?
Android is a **Linux-based open-source operating system** primarily used in smartphones and tablets.  
It was developed by **Google** and is maintained by the **Open Handset Alliance (OHA)**.

### Android Architecture


- **Applications:** Built-in apps like phone, SMS, browser, and third-party apps.  
- **Framework:** APIs used by developers to create applications.  
- **Libraries & Runtime:** Core C/C++ libraries and Android Runtime (ART).  
- **Kernel:** Manages hardware drivers, security, and memory.

### Notable Features
- Multitasking  
- Rich UI & Widgets  
- Open-source flexibility  
- Google Play Store integration  
- Regular updates and security patches
